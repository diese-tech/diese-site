# ADR 0003: Homepage information architecture

## Status

Accepted

## Context

The next portfolio needs a clear hierarchy so employers quickly understand the strongest evidence of frontend/product ability. The project list includes business software, community platforms, booking/payment work, Discord tools, AI experiments, and a future sanitized internal tool.

## Decision

Use this homepage structure:

1. Hero
2. Featured Projects
3. Also Built
4. Build Process
5. Stack
6. About
7. Contact

Use this project order:

### Featured

1. SwiftDispatch
2. Serpent Ascension League
3. ThreeTails Booking

### Also Built

- GodForge
- YapHub
- Atlas
- Sanitized internal tool, later

## Consequences

- The strongest employer-facing projects appear first.
- SwiftDispatch anchors the small-business workflow story.
- Serpent Ascension League demonstrates full app depth, admin systems, testing, and community operations.
- ThreeTails Booking demonstrates client-facing booking/payment workflow work.
- Smaller tools and experiments still add breadth without diluting the homepage's primary argument.
